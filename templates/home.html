Text Feature Engineering for Amazon Product Review Classification

Project Abstract

Sentiment Analysis is a key task in Natural Language Processing that aims to classify opinions expressed in textual data. With the rapid growth of e-commerce platforms, analyzing customer reviews has become essential for understanding user satisfaction and improving products and services. Traditional machine learning approaches often rely on single feature extraction techniques such as Bag of Words or TF-IDF, which fail to capture contextual and semantic information effectively. This project proposes a hybrid sentiment analysis model that integrates both traditional and advanced techniques to improve classification performance on Amazon product reviews. The model combines TF-IDF features with contextual embeddings generated using a pre-trained BERT model to capture both statistical and semantic representations of text. A Convolutional Neural Network (CNN) is employed to extract deep and complex features from the combined feature vectors. Furthermore, an ensemble learning approach using Extra Trees and XGBoost classifiers is applied to enhance robustness and reduce model bias. The dataset used consists of over 568,000 Amazon product reviews, which are preprocessed through text cleaning, normalization, and stopword removal. Sentiment labels are derived from review ratings and categorized into positive and negative classes. The proposed model is trained and evaluated using standard performance metrics including accuracy, precision, recall, F1-score, and ROC-AUC. Experimental results demonstrate that the hybrid model achieves an accuracy of 91.88%, outperforming traditional and standalone approaches. The integration of multiple techniques enables the model to effectively capture both contextual and semantic information, making it suitable for real-world sentiment analysis applications. The proposed system provides a scalable and efficient solution for analyzing large volumes of customer reviews in e-commerce platforms.
